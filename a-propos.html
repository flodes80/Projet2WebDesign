Projet 2 - Réalisation d'un dossier de veille technologique

Ce site web a été réalisé dans le cadre de l'UE Web Design (INFO_31).

Le but de ce site est de présenter un "dossier de veille technologique", ici à propos du framework Bootstrap. Ce projet était à réaliser en groupe d'étudiants, ici 5: COCHEREL Valentin, GARCIA Florian, LECLÈRE Valentin, ROSE Maxime et VOLFF Lorris

Ce travail devait respecter des étapes bien définies pour être mené à bien :

  • Identification des sources d'information
  • Recherches autour de la thématique choisie (collecte d'informations)
  • Analyse des données recueillies
  • Conception d'un dossier synthétique
  • Réalisation d'un site web statique permettant la diffusion conviviale des résultats de cette veille technologique

Gabarit fonctionnel

Nous avons établit un gabarit fonctionnel en amont afin d'avoir une idée de l'allure générale du site. Même si Bootstrap gère différents points de cassure, nous avons réalisé un gabarit fonctionnel pour trois formats d'écran. Les fichiers se trouvent dans le répertoire "assets/illustrator/gabarit-fonctionnel/".
